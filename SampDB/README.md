# SampDB

База данных игроков и информации по серверам Samp-Rp.

## Варианты использования

-   **Десктопное приложение**: `SAMPDB.exe` для Windows.
-   **Скрипт для MoonLoader**: `sampdb.luac` для интеграции в игру.

> Полная инструкция доступна на [сайте](https://amfeeque.github.io/samp.tools/SampDB).
